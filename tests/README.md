# Test Suite Skeleton

This directory provides a starter structure for:
- unit tests
- integration tests
- e2e checks

## Planned tooling
- Vitest for unit/integration
- Supertest (optional) for API route testing
- Playwright (optional) for UI e2e

## Commands (to be expanded)
- `npm run test:unit`
- `npm run test:integration`
- `npm run test:e2e`
